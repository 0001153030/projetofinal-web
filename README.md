# Balança Multiuso

Sistema web para monitoramento de saúde que integra medições de peso, pressão arterial, batimentos cardíacos, temperatura corporal e saturação de oxigênio em uma única plataforma. O projeto simula o funcionamento de uma balança multiuso, permitindo que usuários registrem e acompanhem suas métricas de saúde ao longo do tempo, enquanto administradores gerenciam contas e visualizam dados consolidados.

As funcionalidades incluem cadastro e gerenciamento de medições com dados como peso, pressão sistólica e diastólica, frequência cardíaca, temperatura e SpO₂, além de um painel administrativo protegido por senha para gerenciamento de usuários. O sistema foi desenvolvido com foco em simplicidade de uso e responsividade.

## Tecnologias utilizadas

- **PHP 8.4** com **Laravel 13**
- **Tailwind CSS 4** (via CDN)
- **PostgreSQL** (gerenciado pelo Render)
- **Apache** (servidor web)
- **Docker** (containerização)
